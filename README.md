# SpiderPi Edge-AI Autonomous Robotics System

Edge AI robotics system built on a SpiderPi hexapod for autonomous navigation, real-time computer vision, and IoT environmental monitoring.

**Award:** 1st Place — Robotics Category, IoT Data Hackathon 2026 (Hong Kong)

## Problem

Safety-critical environments need mobile sensing platforms that can perceive hazards on-device, move stably over uneven terrain, and stream monitoring data without depending on a full cloud pipeline.

## Approach

- Real-time computer vision and ML inference on the edge for object / hazard / risk-indicator detection
- Inverse kinematics and gait control for autonomous hexapod navigation
- IoT monitoring workflow connecting onboard sensing to remote status visibility
- Integration of perception, locomotion, and communication into one cyber-physical stack

## Tech stack

- Python
- OpenCV
- TensorFlow
- IoT / robotics control stack (SpiderPi hexapod platform)

## Results

- Demonstrated autonomous navigation with onboard vision-based perception
- Connected sensor/vision outputs to an IoT monitoring workflow
- Won first place in the Robotics category at the IoT Data Hackathon 2026
